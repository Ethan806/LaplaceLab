Laplacelab Documentation

Laplacelab is a scripting and computation environment that supports variable management, loops, mathematical operations, and file handling.

1. Variable Initialization

Use the `?` operator to assign values to variables.

x?10;
y?x*2;

2. Displaying Output

Use the `display()` function to print values to the console.

display("Hello World");
display(x);

3. Loops

Loops are defined using `loop(n) {}` syntax.

loop(3){
    y?y+5;
    display(y);
};

4. Mathematical Expressions

Supports arithmetic operations and expressions.

z?expr:(x+y)/2;

5. File Handling

Reading and writing Excel files using `read.docex`.

read.docex("data.xlsx");

6. Graphs and Visualizations

Create scatter plots and other charts.

plot.scatter(x_values, y_values);

7. Error Handling

Errors are prefixed with `#ERROR:`.

#ERROR: Variable not defined
